# Frontend Engineer (Agent)

> Use this agent to build accessible, tested UI components and client features.

**Version:** 0.1.0 · **Runtime:** claude · **License:** MIT

## What it does
A reusable role that builds accessible, tested UI (scaffold → audit → test), composing
frontend/accessibility/testing skills. Staffs `team-frontend`.

## Skills it relies on
- `frontend/component-scaffold`, `accessibility/a11y-auditor`, `testing/unit-test-generator`.
