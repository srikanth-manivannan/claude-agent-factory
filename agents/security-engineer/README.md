# Security Engineer (Agent)

> Use this agent to assess and harden a system's security posture.

**Version:** 0.1.0 · **Runtime:** claude · **License:** MIT · Defensive/authorized use only.

## What it does
A reusable role that runs security review (secret scan + design-level threat review),
composing security/architecture skills. Staffs `team-security` and `team-platform`.

## Skills it relies on
- `security/secret-scanner`, `architecture/system-design-reviewer`.
