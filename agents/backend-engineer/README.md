# Backend Engineer (Agent)

> Use this agent to implement, persist, and test server-side features from an API contract.

**Version:** 0.1.0 · **Runtime:** claude · **License:** MIT

## What it does
A reusable role that builds server-side features (contract → schema → implementation →
tests), composing backend/database/testing skills. Staffs `team-backend`.

## Skills it relies on
- `backend/openapi-designer`, `database/schema-designer`, `testing/unit-test-generator`.
