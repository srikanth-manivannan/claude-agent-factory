# QA Engineer (Agent)

> Use this agent to verify quality across tests, accessibility, and documentation before release.

**Version:** 0.1.0 · **Runtime:** claude · **License:** MIT

## What it does
A reusable role that owns the quality gate (tests + a11y + docs), composing
testing/accessibility/documentation skills. Staffs every team as the quality role.

## Skills it relies on
- `testing/unit-test-generator`, `accessibility/a11y-auditor`, `documentation/doc-writer`.
