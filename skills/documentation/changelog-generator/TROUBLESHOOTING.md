# Troubleshooting — Changelog Generator

> Symptom → cause → fix. See [SKILL.md](SKILL.md).

## Common issues
### Symptom: the result doesn't match the intended outcome
- **Cause:** unclear or incomplete inputs.
- **Fix:** restate the goal and provide the inputs listed in SKILL.md.

### Symptom: the result conflicts with a project standard
- **Cause:** a relevant constraint wasn't supplied.
- **Fix:** pass the constraint and re-check against [/standards/](../../../standards/).
